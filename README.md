# bebopa

Extension to cache data from rateyourmusic (releases, songs and movies) and glitchwave.

## Installation

### Quick Install (Packaged Extension)

1. **Build the extension packages:**

   ```bash
   ./build-extension.sh
   ```

2. **Install the package for your browser:**
   - **Firefox:** Install `build/rym-cache-overlay-<version>.xpi`
   - **Chrome:** Extract and load `build/rym-cache-overlay-<version>-chrome.zip`

For detailed installation instructions, see [INSTALLATION.md](INSTALLATION.md).

### Developer Installation

See [rym-overlay-extension/README.md](rym-overlay-extension/README.md) for loading the extension directly from source.
